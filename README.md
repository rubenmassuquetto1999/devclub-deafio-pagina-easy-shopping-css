# Easy Shopping Via Mobile - Landing Page

Uma landing page responsiva construída com base em um design do Figma. O projeto apresenta um layout moderno de duas colunas em telas maiores, adaptando-se perfeitamente para uma estrutura empilhada em dispositivos móveis e notebooks menores.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização com Flexbox, posicionamento absoluto/relativo e tipografia (Google Fonts - Oswald). Arquivo de estilo externo.
* **JavaScript (Vanilla):** Manipulação do DOM para controle do estado ativo ("active") dos links de navegação. Lógica separada em arquivo externo para melhor organização.
* **Media Queries:** Responsividade garantida para monitores grandes, notebooks de 13" e smartphones.

## 📱 Funcionalidades e Responsividade

* **Fidelidade ao Design:** Medidas, cores e espaçamentos ajustados milimetricamente para refletir o design original.
* **Layout Desktop:** Estrutura em duas cores (fundo branco e cinza claro) com imagem sobreposta.
* **Layout Notebook (até 1366px):** Reajuste de tipografia e proporções da imagem para evitar quebra de layout em telas intermediárias.
* **Layout Mobile (até 768px):** Transição para layout em coluna única (stacked), com ocultação do menu de navegação e centralização dos elementos principais.
* **Interatividade:** Navegação com feedback visual no clique.

## 🛠️ Como executar o projeto

Como este é um projeto estático, não é necessária nenhuma instalação complexa de pacotes ou servidores.

1.  Certifique-se de que todos os arquivos estão na mesma pasta (`index.html`, `style.css`, arquivo `.js` e a pasta `img` contendo a imagem `Illustration-3-3.png`).
2.  Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador padrão (Chrome, Firefox, Edge, etc.).
3.  Para testar a responsividade, você pode redimensionar a janela do navegador ou usar a ferramenta de inspeção (F12) e ativar o modo de dispositivo móvel.
